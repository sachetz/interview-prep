## Question

Design a data structure to implement N queues using a single array of size S.

It should support the following operations:
1. enqueue(X, M): Enqueues an element X into the Mth queue. Returns true if the element is enqueued into the queue, otherwise false.
2. dequeue(M): Dequeues the top element from Mth queue. Returns -1 if the queue is empty, otherwise, returns the dequeued element.

Constraints:
- 1 <= N <= S <= 1000
- 1 <= X <= 10^5
